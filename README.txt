## Что менять
В index.html у трех кнопок сейчас стоит href="#".
Замени их на свои внешние ссылки.

## Деплой на GitHub Pages
1. Создай репозиторий.
2. Загрузи все файлы из архива в корень репозитория.
3. В GitHub: Settings → Pages.
4. В Build and deployment выбери:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Сохрани.
6. Если нужен домен 212.su, в GitHub Pages укажи Custom domain и добавь DNS-записи у регистратора.
